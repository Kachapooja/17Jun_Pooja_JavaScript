Que-9 In Bootstrap 4, what is flexbox?

 In Bootstrap 4, flex box is used to control the layout and alignment specification of components.

 Flexbox makes easier to design flexible responsive layout structures without using float or positioning attributes.

 In Bootstrap 4, if we need to add display: flex to an element, do so with .d-flex

 In Bootstrap 4, if we need class or display value to allow the use of extra flexbox utilities for sizing, alignment, spacing, and many more.
