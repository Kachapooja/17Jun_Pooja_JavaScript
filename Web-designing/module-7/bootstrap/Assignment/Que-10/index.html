Que-10 How can one create an alert in Bootstrap?

 In Bootstrap Alerts are used to provide an easy way to create predefined alert messages.

 Alert adds a style to our messages to make it more effective to the users.

 Alerts are created with the ".alert" class.

 There are four classes that are used within < div> element for alerts.

  • .alert-success
  • .alert-info
  • .alert-warning
  • .alert-danger

 Example:

Success! You should read this message.
Info! You should read this message.
Warning! You should read this message.
Danger! You should read this message.
