Que-5 What do you know about the Bootstrap Grid System?
  •  Bootstrap Grid system is used to a series of containers, rows and columns to layout and align content.

  •  Bootstrap Grid system is build with the flexbox and it is fully responsive.

  •  Bootstrap Grid system allows up to 12 columns across the page.

  •  In Bootstrap Grid system we don't want to use 12 column indivisually,we can group the column together to create wider column.

  •   Bootstrap Grid system is divided in four classes:

    1. xs - for Portrait mobile screens < 576px
    2. sm - for Landscapes phones screens >= 576px
    3. md - for Tablets screens >= 768px
    4. lg - for small-sized laptop and desktop screens >= 992px
    5. xl - for larger-sizedlaptop and desktop screens >= 1200px

  • Bootstrap Grid system structure:

    1. < div class="container">
     < div class="row">
      < div class="col-xs-6">..write here content..< /div>
      < div class="col-xs-6">..write here content..< /div>
     < /div>
    < /div>
    2. < div class="container">
     < div class="row">
      < div class="col-sm-6">..write here content..< /div>
      < div class="col-sm-6">..write here content..< /div>
     < /div>
    < /div>
    3. < div class="container">
     < div class="row">
      < div class="col-md-6">..write here content..< /div>
      < div class="col-md-6">..write here content..< /div>
     < /div>
    < /div>
    4. < div class="container">
     < div class="row">
      < div class="col-lg-6">..write here content..< /div>
      < div class="col-lg-6">..write here content..< /div>
     < /div>
    < /div>
