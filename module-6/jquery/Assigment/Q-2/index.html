How To Apply CSS in jQuery:


  • →The css() method sets or returns one or more style properties for the selected elements.
    • Syntax:

    $(document).ready(function(){
     $("document").css("propertyname","value");
    });

    Example:

    $(document).ready(function(){
     $(".ans-box").css({"height":"150px","width":"150px","background-color":"brown","border":"1px solid black"});
    });

    How to Add Class and Remove Class in Jquery:


  • →The addClass() method adds one or more class names to the selected elements.

  • →This method does not remove existing class attributes, it only adds one or more class names to the class attribute.

  • →The removeClass() method removes one or more class names from the selected elements.

    • Syntax: (Add Class)

    $(document).ready(function(){
     $('selector').addClass(class_name);
    });

    Example: (Add Class)

    $(document).ready(function(){
     $("#addcls").click(function(){
        $(".rmv").addClass("btn");
    });

    Syntax: (Remove Class)

    $(document).ready(function(){
     $('selector').removeClass(class_name);
    });

    Example: (Remove Class)

    $(document).ready(function(){
     $("#rmvcls").click(function(){
       $(".rmv").removeClass("btn");
    });

    jQuery Animation :


  • →The jQuery animate() method is used to create custom animations.
    • Syntax:

    $(document).ready(function(){
     $(selector).animate({params},speed,callback);
    });

    Example:

    $(document).ready(function(){
     $("#btn").click(function(){
      $("div").animate({
       left:"250px"
      })
     });
    });
