Que-8 How can you use Bootstrap to make thumbnails?

 Bootstrap has an easy way to do this with thumbnails.


 In Bootstrap thumbnails show linked images in a grid structure with minimal required markups.


 Bootstrap allows to create these thumbnails that display linked images in grids with pre-defined classes.


 These classes help to reduce the length of the code.


 Thumbnails help provide a blink of images using smaller versions of the media.


 Syntax:

< div class="row">
 < div class="col-md-12">
  < a href="#" class="thumbnail">
   < img src="..." class="img-thumbnail" alt="">
  < /a>
 < /div>
< /div>
