Que-2 What is a Bootstrap Container, and how does it work?

-->Containers are the most basic layout element in Bootstrap and are required when using default grid system.


-->Bootstrap Container is used to set the contents margins dealing with the responsive behaviors of webpage layout.


-->The Container class is used to create boxed content.There are 2 container class in bootstrap.


  •   1. Container

    -->The container uses '.container' class. It provides a responsive fixed-width container.

  •   2. Container-fluid

    -->Fluid-container uses the '.container-fluid' class. It is used for a full-width container.
