Que-7 What is a Button Group, and what is the class for a basic Button Group?

 "Button Groups" in Bootstrap is a class of name "btn-group" which is used to create a series of buttons in groups.

 Use a < div> element with class ".btn-group" to create a button group.

 This is the basic syntax of the "button group" class where each button has its own class of "btn".

Syntax:

< div class="btn-group">
 < button type="button" class="btn">Click< /button>
< /div>
